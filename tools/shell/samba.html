samba配置

samba

  1. 增加用户

    smbpasswd -a lightxue

  2. 编辑/etc/samba/smb.conf，增加以下内容

    [lightxue]
netbios name = lightxue
valid users = lightxue
path = /home/lightxue
browseable = yes
writable = yes
printable = no
create mask = 0644

  3. 重启samba服务

    /etc/init.d/smbd restart
or
service smb restart

  4. 这样就新建一个samba用户并新建一个目录了

  5. 删除samba密码的缓存

    ; 查看当前的连接
net use
; 删除
net use * /delete
