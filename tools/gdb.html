GDB

gdb

  1. core dump大小不限制

    ulimit -c unlimited

  2. 显示栈帧

    bt full #显示局部变量
bt 3 # 开头3个栈帧
bt -3 # 最后3个栈帧

  3. 显示寄存器eax内容

    p $eax

  4. 格式化输出

    p/x $eax # 十六进制
p/d $eax # 十进制
p/u $eax # 无符号
p/o $eax # 八进制
p/t $eax # 二进制(two)
p/a $eax # 地址
p/c $eax # 字符(ASCII)
p/f $eax # 浮点小数
p/s $eax # 字符串
p/i $eax # 机器语言(仅在显示内存的x命令(eXamining)中可用)

  5. 显示汇编指令

    x/10i $pc # pc所指地址开始的10条指令
disassemble # 当前整个函数
disassemble $pc # 计数器所在函数的整个函数
disassemble $pc $pc+50 # 开始到结束之间的部分

  6. ni(nexti)和si(stepi)逐条招行汇编指令，ni不进入函数体

  7. c(continue)可以加次数作参数

    c 5 # 遇到断点5次不停止

  8. watch <表达式>，表达式指常量或变量等

    watch var   # stop if change(write)
awatch var  # stop if access(read or write)
rwatch var  # stop if read

  9. 删除断点和监视

    i b # info break
d 2 # delete 2

  10. 改变变量的值：set variable <变量>=<表达式>

    set variable var = 10
set variable var = var2 + 10
set variable var = func(100) + 10

  11. 生成core dump，保存当前调试镜像

    generate-core-file filename

  12. 查看进程信息

    info proc

  13. attach到进程

    attach 17606 # attach到pid为17606的进程上
detach # 用完分离

  14. 条件断点

    b main if var == func(argc)
condition 3 if var == func(args) # 给断点3加上触发条件

  15. 忽略断点

    ignore 3 4 # 忽略断点3 4次

  16. 跳出当前代码块

    until
until func # 直到执行完func

  17. 执行完当前函数后暂停

    finish

  18. 反复执行

    continue 次数
step 次数
stepi 次数
next 次数
nexti 次数

  19. clear范围删除断点

    clear func
clear lineno
clear file:lineno
clear file:func
clear breakno

  20. enable, disable启用和禁用断点

  21. 断点命令

    command 2 # 2是断点的编号
>p var # 各种命令
>end

  22. 查看帧信息

    frame    # 查看当前帧
frame 3  # 查看编号为3的帧
i frame 3 # 查看编号为3的帧

  23. 进程内存映射

    i proc mapping

  24. 格式化输出

    printf "%d\n", var

  25. 显示变量

    info variables
